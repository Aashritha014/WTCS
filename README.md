website live at: [Web Technology Case Study](https://wtcs.netlify.app) 

### **Introduction**

- Form validation is an essential part of modern web development. It ensures that users provide accurate and complete data before submission.
- Client-side validation, performed using JavaScript, enhances user experience by providing immediate feedback and reducing the number of invalid submissions sent to the server.
- This case study demonstrates how JavaScript can be used to validate a Hotel/Travel Booking Form, verifying user input for name, email, phone number, dates, and room details.

### **Objective**

- To design and implement a hotel booking form that performs client-side validation using JavaScript, ensuring all entered information is correct, complete, and logically valid before submission.
